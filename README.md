# bot-framework

## Quick start

### Requirements

- Python 3.11
- Create a virtual environment and run:

```bash
pip install -r requirements.txt
```

### User Interface (chat box) for bot testing

- [ui.py (streamlit)](besser/bot/test/ui/ui.py)

```bash
streamlit run besser/bot/test/ui/ui.py
```
